🚀 Setup & Run Instructions

  1. Click the purple manifest.json and content.js buttons below to download both toolkit files.
  2. Create a new empty folder on your computer (e.g., CygnusExtension) and move both downloaded files inside it.
  3. Open your browser, go to chrome://extensions/, enable Developer Mode (top-right), and click Load Unpacked (top-left) to select that folder.
  4. Enter your live Render App URL below, fill the platform and task commands, and hit execute. The system will auto-automate the target page!
